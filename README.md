# Application

Cette application est construite avec :

- **Frontend** : React.js (avec Vite)
- **Backend** : Node.js / phpMyAdmin (via une base de données MySQL)


-- Installation d'un router pour gerer une grosse application 
- `cd /frontend/ -> npm install react-router-dom`
